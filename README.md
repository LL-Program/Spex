## Syntax
 - Declare Function
```
  declf my_function { // declare
    math result1 5+10; // save as variable
    math result2 $result1*2; // calculate with variables
    print $result1 $result2; // print variables
}
my_function; //call
```
 - Declare variable
```
decl object hello! variable1;// decl type data name
```
